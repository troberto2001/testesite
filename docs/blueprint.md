# **App Name**: Golden Roofs

## Core Features:

- Header Navigation: Implement a header with the company logo on the left and a navigation menu on the right with links to Home, Services, About, Testimonials, and Contact.
- Hero Section: Design a hero section with a main title, descriptive subtitle, a large space for a custom image, and a prominent call-to-action button (e.g., 'Solicitar Or√ßamento') in a luxury yellow.
- Services Section: Create cards for each service offered (minimum 3), including a title, description, and space for a custom image. Implement an elegant hover effect on the cards.
- About Section: Develop a section about the company with descriptive text and a space for an illustrative image, using a modern layout divided into image + text.
- Testimonials Section: Design cards featuring a client photo, testimonial text, and their name, with space for custom client images.
- Contact Form: Implement a contact form with fields for Name, Email, and Message, along with a submit button. Use luxury yellow theme colors for the background and buttons.
- Responsive Design: Ensure the landing page adapts perfectly to desktop, tablet, and mobile devices, providing an optimal user experience across all platforms.

## Style Guidelines:

- Primary color: Luxurious Yellow (#FFD700) to convey elegance and professionalism.
- Background color: Desaturated light yellow (#FAF9F0) to provide a subtle contrast and enhance readability.
- Accent color: Golden yellow (#FFC107) for CTAs and highlights, drawing attention to important elements.
- Font: 'Alegreya', a serif font for headings, to project an elegant, intellectual, contemporary feel. For body text also use 'Alegreya' as it is easily readable for long-form paragraphs
- Use minimalist, sophisticated icons related to roofing and cleaning services, ensuring they complement the luxury theme.
- Employ a clean, modern layout with ample white space to provide an uncluttered and sophisticated look. Sections should be clearly delineated, and elements should be well-aligned for ease of navigation.
- Incorporate subtle animations, such as fade-in effects on scrolling or gentle transitions on hover, to enhance the user experience without being distracting.
